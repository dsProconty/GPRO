# GPRO - Gestor de Proyectos Proconty

Sistema de gestión de proyectos de consultoría para Proconty.

## Stack
- **Frontend/Backend:** Next.js 14 + React 18
- **Base de Datos:** PostgreSQL (Neon)
- **ORM:** Prisma
- **Auth:** NextAuth.js
- **UI:** PrimeReact (Sakai)
- **Deploy:** Vercel
